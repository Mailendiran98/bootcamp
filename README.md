## 🚀 Projects

### [PipelinePulse](./pipelinepulse)
**PipelinePulse** is a modern SaaS landing page built as part of this learning repository. It is designed as a simple, business-friendly dashboard to monitor data pipeline health, data freshness, and SLA compliance across modern data stacks (Snowflake, Databricks, Airflow, dbt).

**Tech Stack:**
- **Framework:** Next.js (App Router, TypeScript)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion & Custom CSS (featuring a pure CSS parallax Milky Way starfield)
- **Icons:** Lucide React

**Key Features Highlighted:**
- **Unified Dashboard:** Clear pipeline status (Success, Failed, Delayed) in one place.
- **Freshness & SLA Monitoring:** Instant alerts for stale data or missed SLAs.
- **Lightweight Integration:** Connects via a simple webhook without complex infrastructure changes.
- **Premium UI:** Dark mode, glowing gradients, and responsive layout modeled after high-end SaaS tools.

**How to run locally:**
```bash
cd pipelinepulse
npm install
npm run dev
```
Then open [http://localhost:3000](http://localhost:3000) in your browser.
